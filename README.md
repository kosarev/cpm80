# cpm80
CP/M-80 2.2 emulator with Python API.

Based on the fast and flexible [z80](https://github.com/kosarev/z80) emulator.

## Installing

```shell
$ pip install cpm80
```

## Running and terminating

```
$ cpm80

A>save 1 dump.dat
A>dir
A: DUMP     DAT
A>^C
A>^C
A>
```

Press Ctrl-C three times to exit.
